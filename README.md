## Prompt生成器 ##
提供一些常用的Prompt模板，现在包括
- 防剧透观剧伴侣
- 中老年表情包助手

## 使用方法 ##
1. 点击Chrome插件按钮，展开侧栏
2. 选择想要使用的功能
3. 输入相关信息
4. 生成Prompt


## 功能限制 ##
- 只是个Prompt生成器，无绑定AI，无本地多轮对话。
- 无记忆功能，无数据库功能，无网页抓取功能。
- 只有直接跳转网页Gemini，其他AI请自行打开对话页面。

## 安装方法 ##
未上传Chrome应用商店，请下载release目录中最新的Prompt_Generator.zip，解压后，在chrome://extensions/中打开“开发者模式”（右上角开关），加载未打包的扩展程序（左上角按键），选择之前解压目录后安装。
